## Installation

Install [`nvm`](https://github.com/nvm-sh/nvm) and set your node version to `v12.16.2`:

```sh
nvm use v12.16.2
```

Install the app with `yarn` in this directory (`./fiftyone/app`):

```sh
yarn install
```

Start the app:

```sh
yarn dev
```

That's it!
